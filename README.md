# 🤖 Aprendizaje apoyado con Inteligencia Artificial

Después de varios años sin trabajar nuevamente con Arduino y electrónica básica, este proceso representó una oportunidad para retomar el aprendizaje práctico utilizando componentes que anteriormente no había probado.

Gracias al apoyo de la Inteligencia Artificial, fue posible comprender:

* conexiones electrónicas,
* lógica de programación,
* uso de librerías,
* configuración de sensores,
* resolución de errores,
* y funcionamiento general de distintos módulos.

La IA sirvió como guía durante cada prueba realizada, permitiendo avanzar paso a paso en la construcción de circuitos y en el desarrollo del código necesario para hacer funcionar cada componente correctamente.

Entre las pruebas realizadas se encuentran:

* Pantalla LCD 16x2 con I2C
* Sensor ultrasónico HC-SR04
* Integración LCD + sensor de distancia
* Configuración del módulo Bluetooth HC-05
* Comunicación serial y monitor serial

Además de ayudar a resolver errores de compilación y conexión, este proceso permitió reforzar conocimientos de programación y electrónica que no se practicaban desde hace varios años.

Este repositorio representa no solo pruebas técnicas, sino también un proceso de reaprendizaje y exploración tecnológica utilizando herramientas modernas de apoyo como la Inteligencia Artificial.
